Illustration of the Encoding/Decoding Process

            

                
1. Text to Binary

                
Each character in the text is converted to its ASCII value, which is then transformed into an 8-bit binary string. Example:

                
Character: A
ASCII: 65
Binary: 01000001
                

            

            

                
2. Binary to DNA

                
The binary string is split into pairs of two bits and each pair is mapped to a DNA base:

                

                    
                        

                            Binary Pair
                            DNA Base
                        

                    
                    
                        
00A

                        
01C

                        
10G

                        
11T

                    
                

                
Binary: 01000001
DNA: CAG
                

            

            

                
3. DNA to Binary and Decoding

                
The DNA sequence is converted back to a binary string, which is then grouped into 8-bit segments to decode back to characters:

                
DNA: CAG
Binary: 01000001
Character: A
